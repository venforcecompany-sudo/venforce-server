Como usar
1
Baixe o arquivo .zip clicando no botão acima.
2
Extraia a pasta no seu computador.
3
Instale o Python caso ainda não tenha (python.org).
4
No terminal, dentro da pasta extraída, instale as dependências:
pip install playwright
5
Em seguida rode:
playwright install
6
Execute o script principal:
python v1_10_1.py
7
Na primeira execução, faça login no Mercado Livre quando a janela abrir.
8
A sessão será salva automaticamente para os próximos usos.
